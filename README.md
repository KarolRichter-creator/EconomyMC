
EconomyMC is an economy and land-management mod for NeoForge that adds an integrated economy, plot, market, check, and jobs system to your server.

With EconomyMC, players can manage their plots, sell land and items, create market listings, accept jobs, and earn money through the built-in economy system. In addition, the mod provides daily rewards, a treasury and tax system, checks, server administration features, and special areas such as the capital.

**Features**

Plot System
- Manage your own plots
- Support for normal claims and capital claims
- Claim credits for different plot types
- Market system for selling plots

**Market & Shop**
- Browse plots and offers on the market
- Manage your own sales
- In-game shop menus
- Direct selling and buying flows

**Jobs System**
- Create, accept, and confirm jobs
- Support for server-created jobs
- Rewards for completed tasks
- Configurable start times for jobs

**Economy System**
- Player balances and money management
- Treasury system for the server
- Daily rewards
- Tax and fee system
- Automatic or manual tax adjustment
- Scoreboard support

**Checks**
- Create checks directly through the menu
- Redeem checks through chat code input
- Integrated voucher-style money system
